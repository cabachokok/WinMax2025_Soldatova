# WinMax2025_Soldatova

МОДУЛИ:

main - пользовательский блок. Выбор настроек для создания тайлсета (палитры, размера и уровня детализации), вызов сохранения тайлсета

generator - функции генерации единичного тайла и тайлсета. Хранение цветов палитр

saved - Сохранение тайлсета в формате png
